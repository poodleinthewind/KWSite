Features

Everything you need, built in.

DropComp turns a folder of .aep assets into a browseable, searchable library right inside After Effects.
⚡

Instant Preview

Hover over any asset to see an animated preview without importing anything into your project.

📂

Organized Categories

Assets are sorted into transitions, text animators, title cards, logos, and more — all browsable from the sidebar.

⭐

Favorites

Star the assets you use most and access them from a dedicated favorites panel for quick workflows.

🔄

Multi-Library

Switch between multiple asset folders using the library dropdown. Different projects, different clients.

🎛️

Adjustable Grid

Toggle between S, M, and L grid sizes to see more assets at once or get larger previews.

🛠️

Admin Tools

Generate hover previews for your assets individually or in batch with the built-in admin panel.

Installation

Up and running in 2 minutes.

No terminal. No complicated setup.

Install the Extension

Double-click DropComp.zxp or drag it onto the ZXP Installer app.

Restart After Effects

Quit and relaunch After Effects so it registers the new extension.

Open DropComp

Go to Window → Extensions → Drop Comp to open the panel.

Set the Kit Location

Click Kit Location in the sidebar and point it to your asset folder.

~/Desktop/My Motion Assets

Start Creating

Browse assets, hover to preview, double-click to apply. That's it.
